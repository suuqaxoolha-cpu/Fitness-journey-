# Fitness & Nutrition Tracking App 🚀

**Live Demo:** [https://netlify.app](https://peppy-salmiakki-1cb4ee.netlify.app/)

A comprehensive full-stack application designed to help users track their workouts and manage their daily nutrition effectively. This project showcases advanced frontend logic and a modern user interface.

## 🛠 Tech Stack
*   **Frontend:** React.js & TypeScript
*   **Styling:** Tailwind CSS
*   **Build Tool:** Vite
*   **Deployment:** Netlify

## ✨ Key Features
*   **User Authentication:** Secure Login and Sign-up system.
*   **Workout Management:** Track various exercises (Push-ups, Squats, Cardio) with duration and calorie burn estimates.
*   **Nutrition Tracking:** Log daily meals (Breakfast, Lunch, etc.) and monitor calorie intake.
*   **Dynamic Dashboard:** Visual charts showing weekly activity, steps, and goal progress.
*   **Responsive UI:** Fully optimized for mobile devices, featuring both Dark and Light modes.
